## installation

### R

```R
install.packages('devtools')
devtools::install_github('JiekaiLab/dior')
# devtools::install_github('JiekaiLab/dior@HEAD')
```

### Python

```shell
git clone https://github.com/JiekaiLab/diopy.git
cd diopy
python setup.py install
```
